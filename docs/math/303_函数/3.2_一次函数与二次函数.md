# 3.2 一次函数与二次函数

## 3.2.1 二次、二次问题

解二次函数一般使用 [**配方法（Completing the square）**](https://baike.baidu.com/item/%E9%85%8D%E6%96%B9%E6%B3%95/7600853)

## 3.2.2 一次函数模型

函数 ***`y=kx+b (b≠0, x∈R)`*** 叫做 **一次函数**。

**函数值的改变量与相应自变量的改变量成正比。**

## 3.2.3 二次函数模型

函数 ***<code>y=ax<sup>2</sup>+bx+c (a≠0)</code>*** 叫做 **二次函数**。
它的 *定义域* 是 **`R`**。
