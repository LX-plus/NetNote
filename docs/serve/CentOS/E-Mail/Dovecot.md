# Dovecot

::: tip
Dovecot 是一个用于 Linux/UNIX 系统的开源 IMAP 与 POP3 电子邮件服务器，主要考虑到安全性。
Dovecot 是小型与大型部署的绝佳选择。
它快速，易于设置，不需要特殊的管理，只使用非常少的内存。

[官网](https://www.dovecot.org/)
[官方文档](https://doc.dovecot.org/)
:::
