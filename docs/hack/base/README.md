# 获取工具

## Kali Linux

::: tip
Kali Linux 是最先进的用于渗透测试的 Linux 发行版

Kali Linux 是一个开源，基于 Debian 的 Linux 发行版，用于各种信息安全任务，例如：穿透测试、安全研究、计算机取证与逆向工程。

[官网](https://www.kali.org/)
| [官方文档](https://www.kali.org/docs/)
| [工具文档](https://www.kali.org/tools/)
:::
